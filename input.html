ระบบกรอกข้อมูล — การแก้ไขปัญหาบุกรุกที่ดิน/ป่าไม้

จังหวัดหนองบัวลำภู · เข้าสู่ระบบด้วยรหัสเจ้าหน้าที่

เพิ่มรายการใหม่

กรอกแล้วกด "บันทึก" — หากเลือกแก้ไขรายการเดิม ระบบจะอ่านค่าล่าสุดมาให้แก้ได้ทันที

รายการทั้งหมด (0)

คลิก "แก้ไข" เพื่ออ่านค่าล่าสุดมาแก้ในฟอร์มด้านบน
อำเภอประเภทเรื่องสถานะอัปเดต

นำเข้าข้อมูลจากไฟล์ Excel

ใช้แม่แบบ (Template) ที่ให้ไว้ — ถ้ามีคอลัมน์ id ตรงกับรายการเดิม ระบบจะอัปเดตทับ ถ้าไม่มีจะเพิ่มใหม่
📤 คลิกเพื่อเลือกไฟล์ .xlsx หรือ .csv (หรือลากไฟล์มาวาง)
คอลัมน์ที่รองรับ: id(ไม่บังคับ), fiscalYear, district, type, caseTitle, actionTaken, status, patrolTarget, patrolActual, caseBacklog, resolvedPercent, responsibleOrg

ส่งออกข้อมูลเป็น Excel

ดาวน์โหลดข้อมูลปัจจุบันทั้งหมด แก้ไขในเครื่อง แล้วนำกลับมานำเข้าใหม่ได้ที่แท็บ "นำเข้าจาก Excel"

สถิติจับกุมคดีบุกรุกป่าไม้ รายปีงบประมาณ

ใช้สำหรับกราฟเปรียบเทียบย้อนหลังบนหน้าแดชบอร์ดสาธารณะ
ปีงบประมาณ (พ.ศ.)จำนวนคดี

ผู้ประสานการปฏิบัติ ข้อมูลนี้จะไม่แสดงบนหน้าแดชบอร์ดสาธารณะ

ชื่อ-สกุล / ตำแหน่ง / หน่วยงาน / เบอร์โทรศัพท์ — เข้าถึงได้เฉพาะผู้มี PIN เท่านั้น
ชื่อ-สกุลตำแหน่งหน่วยงานเบอร์โทร
